# 🔥 GOD MODE DDoS - Termux Edition

## 🚀 **ОДНА КОМАНДА ДЛЯ УСТАНОВКИ**
# 1. Обновить пакеты
pkg update -y && pkg upgrade -y

# 2. Установить Python и Git
pkg install python git -y

# 3. Скачать этот репозиторий
git clone https://github.com/55mir2812-sketch/god-ddos.git

# 4. Зайти в папку
cd god-ddos

# 5. Установить библиотеки
pip install aiohttp aiohttp-socks aiofiles asyncio uvloop colorama pystyle fake-headers requests[socks] PySocks httpx beautifulsoup4 lxml

# 6. Запустить атаку
python termux_god_ddos.py
